Introdução ao CSS

O que é o CSS?

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma
linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. Pense na
decoração da sua página. Utilizando o CSS é possível alterar a cor do texto e do fundo, fonte e
espaçamento entre parágrafos. Também pode criar tabelas, usar variações de layouts, ajustar imagens
para suas respectivas telas e assim por diante.

A relação entre HTML e CSS é bem forte. Como o HTML é uma linguagem de marcação (o alicerce de um
site) e o CSS é focado no estilo (toda a estética de um site), eles andam juntos.CSS não é
tecnicamente uma necessidade, mas provavelmente você não gostaria de olhar para um site que usa apenas HTML.
Vejamos abaixo 10 dos inúmeros atributos do CSS.

ATRIBUTOS DO CSS O QUE ELE CONFIGURA
Background-position Posicionamento da imagem de fundo
Border Largura, estilo e cor de todas as 4 bordas
Cursor Tipo de ponteiro do mouse
Font-family Tipo de fonte
Padding Espaço em torno de um elemento em todos os lados
Page-break-after Inserir quebra de página depois de um elemento
Position Como o elemento é posicionado na página
Text-transform Transformação para todas maiúsculas, minúsculas ou inicial maiúscula
Visibility Se elemento é visível ou invisível
Z-index Posição do elemento na pilha
