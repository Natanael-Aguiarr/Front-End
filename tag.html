  1. O que é a tag div?

    • A tag div é uma abreviação para “divisão”. Ela não representa nada em si, mas serve como um container genérico para agrupar e organizar conteúdo em partes específicas de uma página da web.

    • Basicamente, a div é uma caixa na qual você pode colocar outros elementos HTML que pertencem a um mesmo grupo ou que precisam ser estilizados de forma semelhante.

  2. Exemplo de uso da tag div:

    3. <body> <div style="background-color:#00BFFF;text-align:center"> <p>Primeira Div</p> </div> <div style="border:3px dotted #9F0"> <p>Segunda Div</p> </div> </body>

  1. O que é a tag span?

    • A tag span é um container genérico em linha para conteúdo fraseado, mas não representa nada por natureza.

    • É muito parecido com o elemento div, mas enquanto o div é um elemento de nível de bloco, o span é um elemento em linha.

  2. Exemplo de uso da tag span:

    3. <body> <p>Este é um parágrafo de texto normal com uma <span>palavra</span> destacada.</p> </body>
  1. O que é a tag img?

    • A tag img representa a inserção de uma imagem no documento HTML.

  2. Exemplo de uso da tag img:

    3. <body> <img src="https://cdn.pixabay.com/photo/2024/02/16/19/22/green-parrot-8578205_1280.jpg" alt="periquito" </body>
  1. O que é a tag aside?

    • A tag aside representa uma seção de uma página que consiste de conteúdo tangencialmente relacionado ao conteúdo do seu entorno, que poderia ser considerado separado do conteúdo principal.

    • Essas seções são frequentemente representadas como barras laterais ou caixas de destaque.

  2. Exemplo de uso da tag aside:

    3. <body> <aside> <p>Algum conteúdo relacionado a um <article>.</p> </aside> </body>
  1. O que é a tag audio?

    • A tag audio é utilizada para embutir conteúdo de som em um documento HTML.

    • Com o elemento audio, você pode incorporar arquivos de áudio em suas páginas da web.

  2. Exemplo de uso da tag audio:

    3. <body> <audio controls loop> <source src="drago.m4a" type="audio/mp3"> </audio> </body>
  1. O que é a tag canvas?

    • É um elemento HTML que permite desenhar gráficos usando scripts, geralmente em JavaScript. Ela oferece uma área de desenho onde você pode criar composições de fotos, animações e outros elementos visuais.

    • Além disso, é possível incluir conteúdo alternativo dentro do bloco canvas, que será renderizado mesmo em navegadores antigos ou quando o JavaScript está desabilitado.

  2. Exemplo de uso da tag canvas:

    3. <body> <canvas id="meuCanvas" width="300" height="150"> Desculpe-me, seu navegador não suporta o elemento <canvas>. </canvas> </body>
  1. O que é a tag datalist?

    • A tag datalist é um elemento HTML que contém um conjunto de elementos option representando as opções possíveis para o valor de outros controles.

    • Ela é frequentemente usada para fornecer uma funcionalidade de autocompletar em elementos input. Quando os usuários inserem dados em um campo de entrada, eles veem uma lista suspensa com opções pré-definidas.

  2. Exemplo de uso da tag datalist:

    3. <body> <input list="browsers" /> <datalist id="browsers"> <option value="Chrome"></option> <option value="Firefox"></option> <option value="Internet Explorer"></option> <option value="Opera"></option> <option value="Safari"></option> </datalist> </body>
  1. O que é a tag details?

    • A tag details é usada em HTML para criar um elemento que permite ao usuário obter informações adicionais.

  2. Exemplo de uso da tag details:

    3. <body> <details> <summary>Alguns detalhes</summary> <p>Mais informações sobre os detalhes.</p> </details> </body>

  1. O que é a tag embed?

    • A tag embed é um elemento HTML que permite incorporar conteúdo externo em um documento da web.

    • Esse conteúdo pode ser fornecido por um aplicativo externo ou outra fonte interativa, como um plug-in de navegador.

  2. Exemplo de uso da tag embed:

    3. <body> <embed type="video/quicktime" src="movie.mov" width="640" height="480" /> </body>
  1. O que é a tag nav?

    • A tag nav é um elemento HTML que representa uma seção de uma página com links de navegação.

    • Ela aponta para outras páginas ou áreas dentro da própria página.

  2. Exemplo de uso da tag nav:

    3. <body> <nav> <ul> <li><a href="#">Página inicial</a></li> <li><a href="#">Sobre</a></li> <li><a href="#">Contato</a></li> </ul> </nav> </body>
  • Página inicial
  • Sobre
  • Contato
  1. O que é a tag output?

    • A tag output é um elemento HTML utilizado para exibir os resultados de um cálculo ou o resultado de uma ação do usuário.

    • Basicamente, ela permite que um site ou aplicativo injete informações na página conforme necessário.

  2. Exemplo de uso da tag output:

    3. <body> <form oninput="result.value=parseInt(a.value)+parseInt(b.value)">0 <input type="range" name="b" value="50" />100 + <input type="number" name="a" value="10" /> = <output name="result"></output> </form> </body>
0 100 + =
  1. O que é a tag progress?

    • A tag progress é um elemento HTML usado para visualizar o progresso de uma tarefa.

    • Geralmente, ele é exibido como uma barra de progresso.

  2. Exemplo de uso da tag progress:

    3. <body> <progress value="70" max="100">70 %</progress> </body>
70 %
  1. O que é a tag video?

    • A tag video é um elemento HTML utilizado para incorporar conteúdo de vídeo em um documento HTML.

    • Ela permite que você exiba vídeos diretamente na sua página da web.

  2. Exemplo de uso da tag video:

    3. <body> <video controls autoplay> <source src="bomb.mp4" type="video/mp4"> </video> </body>
  1. O que é a tag ul?

    • A tag ul é um exemplo de tag semântica em HTML.

    • Ela é utilizada para criar listas não ordenadas ou desordenadas.

  2. Exemplo de uso da tag ul:

    3. <body> <ul> <li>primeiro item</li> <li>segundo item</li> <li>terceiro item</li> </ul> </body>
  • primeiro item
  • segundo item
  • terceiro item

    • Ela é usada para representar um item que faz parte de uma lista.

    3. <body> <ol> <li>Maça</li> <li>Banana</li> <li>Laranja</li> </ol> </body>
